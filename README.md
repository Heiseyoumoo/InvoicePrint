# InvoicePrint

# 发票合并工具

一个简洁、专业、无需登录的发票合并网站。用户可以上传多个电子发票或扫描件，选择每页拼贴数量，将多张发票自动排版并合并为一个 PDF 文件，支持实时预览、下载和打印。采用纯前端本地处理方案，不依赖后端服务，不需要用户登录，不将发票文件上传到服务器，尽可能保护用户隐私。

## 项目简介

本项目主要用于解决日常报销、财务整理、发票归档过程中，多个发票文件分散、不方便统一打印和提交的问题。

用户只需要上传 PDF 或图片格式的发票，系统即可自动将发票按照每页 2 张、3 张或 4 张的方式进行排版，生成新的合并 PDF 文件。

##  隐私保护

本工具设计为本地处理模式：

* 无需登录
* 不上传服务器
* 不保存用户文件
* 不留存发票数据

页面保留隐私声明：

```text
隐私声明：本工具无需登录，所有处理均在本地完成，我们不会在服务器留存您的任何发票数据。
```

### PDF 处理

使用：

* `pdf-lib`：用于创建、合并、排版 PDF
* `pdf.js`：用于浏览器端 PDF 预览

## 本地运行

如果是纯静态页面，可以直接打开

```text
Invoice.html
```
